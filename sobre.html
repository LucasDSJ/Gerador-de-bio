O que é o Gerador de Bio?

Este projeto tem como objetivo simplificar a criação de bios para redes sociais, portfólios e perfis profissionais. Através de uma interface intuitiva e moderna, o usuário pode gerar descrições automáticas com base em diferentes estilos e temas.
